About PyCam:

PyCam was started out of the idea that i wanted to cheaply install a security camera system in my home. the costs were reasonable for a software developer however i felt this should be more accessable to more people.

PyCam is a simple django website. it was a nice pet project to get me thinking about new tech and hardware integration over wide area network. designed in mind to be hosted in a cloud network so that when recording your home the data will be safely stored outside your home so the intruders do not damage the footage. Additionally it can be easily managed and maintained without proprietary systems.
Out of the gate you can use any old ip cam. But you can also use a rasbery pi webcam if you so desire as well. Overall its just better than a store bought system with roughly the same time investment.
